## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:cultab/dotfiles ~/.config/cultab/dotfiles
NVIM_APPNAME=cultab/dotfiles/neovim/.config/nvim nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=cultab/dotfiles/neovim/.config/nvim nvim
```
